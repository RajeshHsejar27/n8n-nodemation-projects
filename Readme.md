# n8n-nodemation-projects

A collection of production-ready n8n workflows designed to automate business processes using Generative AI, API integrations, and low-code logic.

## 📂 Repository Structure

Each folder in this repository is organized as follows:
* **Workflow JSON:** The raw `.json` export for direct import into your n8n instance.
* **Documentation:** Screenshots of the workflow architecture and successful execution results.
* **Sample Data:** Reference CSV/Excel files used for testing the automation.

---


## 🛠️ Getting Started

1. **Import:** Download the `.json` file from any project folder. In n8n, click **Import from File**.
2. **Credentials:** Set up your API credentials for OpenAI (or Groq/Ollama), Google Sheets, and Gmail.
3. **Regex Setup:** For text-heavy AI outputs, ensure the Code nodes are used to clean strings before sending data to the final destination.
